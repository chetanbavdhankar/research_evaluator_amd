# SpecCurve L0 AMD MI300X Evidence Lab

Static Hugging Face Space for the Wiki C L0 public demo.

This Space renders precomputed JSON/SVG artifacts only. It has no server-side execution,
no user uploads, no user-supplied code path, and no backend API dependency.

When packaged after the AMD run, the Space includes `data/benchmark.json`,
`data/benchmark.json.sha256`, and `methodology/hardware.log` as the MI300X evidence bundle.
