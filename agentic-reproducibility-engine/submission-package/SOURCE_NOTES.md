## AMD/Hugging Face Context

The AMD Developer Hackathon page positions Track 1 around AI agents and agentic workflows, with AMD Developer Cloud, ROCm, Hugging Face, and Qwen as relevant technologies. It also asks participants to publish the completed project as a Hugging Face Space and submit the Space link on lablab.

## Build In Public Challenge

The same page describes an optional build-in-public challenge:

- Share at least two technical updates on social media.
- Provide meaningful feedback about ROCm, AMD Developer Cloud, or APIs.
- Open-source the project or publish a technical walkthrough.

Drafts for those deliverables are in `BUILD_IN_PUBLIC.md`.
